# Commands

## Plugin Manager

| Command | Description |
|---|---|
| `:Lazy` | Open Lazy.nvim dashboard |
| `:Lazy sync` | Install/update all plugins |
| `:Lazy update` | Update all plugins |
| `:Lazy clean` | Remove unused plugins |
| `:Lazy check` | Check for plugin updates |
| `:Lazy profile` | Show startup profiling |
| `:Lazy health` | Run health checks |
| `:Lazy debug <plugin>` | Debug a specific plugin |
| `:Lazy log` | Show Lazy.nvim log |

## LSP

| Command | Description |
|---|---|
| `:LspInfo` | Show LSP client status |
| `:LspLog` | Open LSP log |
| `:LspStart <server>` | Start an LSP server |
| `:LspStop <server>` | Stop an LSP server |
| `:Mason` | Open mason.nvim package manager |

## File Management

| Command | Description |
|---|---|
| `:Oil` | Open oil.nvim file explorer |
| `:update` | Save current buffer |
